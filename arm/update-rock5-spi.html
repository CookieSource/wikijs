Update SPI flash on Radxa Rock 5B

In order to prepare the Rock5B a few initial steps need to be performed.  These steps are needed in order to ensure a clean image write to the target, and ensure the device is able to use the image file.

  • Obtain the boot image file from the Radxa link below.
    wget https://dl.radxa.com/rock5/sw/images/others/zero.img.gz
  • Extract the boot image.
    gzip -d zero.img.gz
  • Write the boot image to the Flash memory or storage medium using the linux dd command.
    sudo dd if=zero.img of=/dev/mtdblock0
