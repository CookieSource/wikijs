RebornOS ARM is now a separate project, led by BredOS. RebornOS focuses on x86_64, while BredOS specializes in ARM-based Linux systems. RebornOS ARM users will be upgraded to BredOS with new branding.

We offer a lot of features during installation, they'll be listed by category.

Devices

Currently we offer images for:

  • Orange Pi 5
  • Radxa Rock 5B
  • Raspberry Pi 4/400
  • Mekotronics R58X

Coming soon:
