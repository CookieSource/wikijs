How to install Hardinfo

You can install Hardinfo with pacman

sudo pacman -S hardinfo

How to use Hardinfo

Opening Hardinfo opens the system information screen.

Here you click click through the different categories to get more info on them.

Clicking Copy to clipboard copies the information of the current page to your clipboard to paste elsewhere.

Generate a report

Clicking generate a report opens a new window where you can select which system information to include.

Clicking generate allows you to save that info as a file for easy sharing.
